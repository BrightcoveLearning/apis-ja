WebVTT ファイルが無効です

Dynamic Ingest API を使用して無効な WebVTT ファイルを取り込むと、操作は成功し、ファイルはアセットとして動画に追加されます。ただし、キャプションやチャプターは Brightcove Player では機能せず、動画に関連付けられた他の有効な WebVTT ファイルも動作しません。

対処策

このオンラインバリデータを使用して WebVTT ファイルを検証できます。
