カスタムフィールド

フィールド タイプ 説明
max_custom_fields 数値 パブリッシャーがアカウントで作成できるカスタムフィールドの数
custom_fields 配列 アカウントのカスタムフィールドオブジェクトのリスト
standard_fields 配列 必須フィールドまたは説明があるフィールドのリスト。どちらも持っていない標準フィールドは、この配列には表示されません。

カスタムフィールドマップ

各カスタムフィールドについて:

フィールド タイプ 説明
id 文字列 フィールドの内部名
description 文字列  
display_name 文字列  
type 文字列 '列挙型の1つ' または '文字列' （これは 'リストに変換されます' または 'テキスト' UIで）
enum_values 配列 カスタムフィールドに有効な文字列値の配列。型= 'enum' の場合にのみ存在します
required ブール値 ビデオを有効にする前に、このフィールドに値が必要かどうか
