400 BAD_REQUEST このエラーは、リクエストにクライアント側のエラーがあったことを示します。メッセージには、通常、エラーの内容に関する情報が含まれています。 ザ・message応答のフィールドには、「無効な値」など、エラーの原因に関する情報が含まれています。sortパラメータ。
400 UNSUPPORTED_FIELD_COMBINATION_ERROR このエラーは、fieldsレポートのパラメータに、レポートタイプに無効な1つ以上のフィールドが含まれていました。 ザ・message応答のフィールドには、指定された無効なフィールドに関する情報が含まれています。
400 TIME_RANGE_UNAVAILABLE この応答は、履歴/バッチ処理システムに要求しているデータがない場合に返されます。
401 UNAUTHORIZED このエラーは、作成に使用された認証（OAuth2 または LA トークン）がリクエストが有効でないことを示します。 使用されている認証トークンを確認します。使用されたトークンが有効だったタイムウィンドウを超えているため、新しいトークンが必要である可能性があります。通常、各トークンは 5 分後に有効期限が切れます。承認トークンは、urlパラメーターとして指定できますaccess_tokenまたはでAuthorizationヘッダ。参照するAPI資格情報の管理詳細については。
404 NOT_FOUND このエラーは通常、要求されたエンドポイントが存在しない場合に送り返されます。 要求されているエンドポイントを確認します。
405 METHOD_NOT_ALLOWED このエラーは、APIリクエストが以外のHTTPメソッドで行われた場合に発生しますGET。 APIへのすべてのリクエストをHTTP GET。
500 SERVER_ERROR このエラーは、サーバーに内部エラーがある場合に発生します。 サーバーが使用できない場合、または一時的な問題が発生している場合は、これを返送することがあります。ある程度の指数バックオフでしばらくしてからもう一度試してください。問題が解決しない場合は、サポートチケットを作成してください。
503 PROCESSING これは、サーバーが要求を受信し、現在それを処理していることを意味します。 NS Analytics API長時間実行されるクエリが発生した場合、このメッセージを送り返す可能性があります。クエリが終了すると、サーバーのキャッシュに最大 5 分間保存されます。したがって、このエラーを受け取ってから4分後にAPIを照会することをお勧めします
