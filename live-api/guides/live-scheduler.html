入門

ライブスケジューラを使用すると、次のアクションをスケジュールできます。静的エントリポイント（SEP）ジョブ。。時間でスケジューラワークフローを設定するだけで、必要なAPIアクションが自動的に処理されます。次のセクションで詳しく説明するように、設定できるスケジューラワークフローには2種類あります。

スケジューラーの開始/停止

スケジューラーを使用すると、SEPジョブのアクティブ化時間と非アクティブ化時間を設定できます。また、時間を更新することもできます。詳細については、をご覧ください。 SEPストリームのアクティブ化/非アクティブ化のスケジューリング。

クリップスケジューラ

ライブジョブからのクリップの生成をスケジュールし、後でスケジュールを更新したり、何かが変更された場合にクリップをキャンセルしたりすることもできます。詳細については、をご覧ください。ライブストリームのクリップ生成のスケジュール。
