サンプルに関する注意

このサンプルのコードはすべてJavaScriptで、2つの部分があります（どちらもCodePenにあります：プラグインスクリプトは上部にあり、下部にはプラグインをプレーヤーに追加する短いコード）。

  • プラグインをvideojsプレーヤーに追加するスクリプト
  • データコレクターにデータを送信するプラグインのスクリプト

データがデータ収集 API に「送信」される方法は、リクエスト URL srcを新しいイメージの値として設定することです（img）。要素を選択し、その要素をページに注入します。

ではなく、同じ方法を使用する必要がありますXMLHttpRequestまたは回避するために他のいくつかCORS問題。
