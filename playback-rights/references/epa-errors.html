この機能は、Playback Rights ManagementServiceの限定可用性フェーズにアクセスできる特定のお客様にご利用いただけます。

エラーリファレンス

次のステータスリストとエラーコードは、Brightcove の再生権限管理サービスで作業するときの問題をデバッグするのに役立ちます。

ステータス エラーコード エラーサブコード 説明
400 BAD_REQUEST DUPLICATE_PARAMETERS 同じパラメータ名がリクエストで複数回指定されました。
401 ACCESS_DENIED TOKEN_REQUIRED NS JSONWebトークン（JWT）は再生権で必要ですが、提供されていません。
401 ACCESS_DENIED TOKEN_VALIDATION NS JSONWebトークン（JWT）提供されたものは検証に合格しませんでした。
403 FORBIDDEN ACCOUNT_ID ポリシーキーのアカウント ID が API リクエストのアカウントと一致しません。
403 FORBIDDEN CLIENT_GEO 動画は現在の地域での再生が制限されています。メッセージには、特定の問題に関する追加情報が含まれます。
403 FORBIDDEN CLIENT_IP 動画は、現在の IP アドレスで制限されています。
403 FORBIDDEN DOMAIN 動画は現在のドメインで再生できないように制限されています。
403 FORBIDDEN OFFERS NS JSONWebトークン（JWT）は、ビデオで利用可能なオファーに基づく再生を許可していません。
403 FORBIDDEN TAGS NS JSONWebトークン（JWT）は、ビデオで使用可能なタグに基づく再生を許可していません。
403 FORBIDDEN VIDEO_NOT_PLAYABLE 1 つのビデオリクエストの場合、ビデオは存在しますが、現在は再生できません。
404 NOT_FOUND VIDEO_NOT_FOUND 要求されたビデオは利用できません。
404 NOT_FOUND PLAYLIST_NOT_FOUND 要求されたプレイリストは使用できません。
405 METHOD_NOT_ALLOWED この API では、GET、HEAD、および OPTIONS のみが許可されます。
500 SERVER_ERROR 内部サーバーエラー。
502 SERVER_ERROR ACCOUNT_RETRIEVE_FAILURE バックエンドサーバから不正な応答を受信しました。
502 SERVER_ERROR VIDEO_RETRIEVE_FAILURE
502 SERVER_ERROR VIDEO_URLS_RETRIEVE_FAILURE
502 SERVER_ERROR PLAYLIST_RETRIEVE_FAILURE
502 SERVER_ERROR PLAYBACK_RIGHT_RETRIEVE_FAILURE CDN から直接再生を取得できません。
502 SERVER_ERROR PLAYLIST_VIDEOS_RETRIEVE_FAILURE
502 SERVER_ERROR LICENSE_RETRIEVE_FAILURE このビデオのライセンスサーバーからライセンスを取得できませんでした。
502 SERVER_ERROR OFFERS_RETRIEVE_FAILURE この動画の API からのオファーを取得できませんでした。
502 SERVER_ERROR RIGHTS_RETRIEVE_FAILURE このビデオの再生権限を取得できませんでした。
503 SERVICE_UNAVAILABLE バックエンドサーバからこの応答を返しました。
504 SERVER_TIMEOUT バックエンドサーバまたはそれらが依存しているサーバの 1 つがタイムアウトしました。
