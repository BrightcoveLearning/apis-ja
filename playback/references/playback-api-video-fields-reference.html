主フィールド

表1:[主フィールド]
フィールド タイプ デフォルト 説明
id 文字列 なし 動画ID
account_id 文字列 なし アカウントのID
created_at ISO 8601 日付時刻文字列 なし ビデオがアカウントに追加された日時。例：「2014-12-09T 06:07:11 .877Z」
cue_points の配列cue_pointオブジェクト 空の配列 ミッドロール広告リクエストまたはプレーヤーAPIを介して作成されるその他のアクションのマーカー
custom_fields オブジェクト 空のオブジェクト カスタムフィールド名のマップ：値のペア;値を持つフィールドのみが含まれます
description 文字列 ヌル ビデオの簡単な説明-最大250文字のシングルバイト文字
duration 数値 ヌル ビデオの長さ（ミリ秒）
economics 文字列 AD_SUPPORTED 動画の広告リクエストが許可されているかどうかを示します。値：AD_SUPPORTEDまたはFREE
labels 文字列の配列 ヌル の配列ラベルビデオに割り当てられました
link リンクオブジェクト ヌル 関連リンク
long_description 文字列 ヌル 最大5000バイトの文字
name 文字列 なし ビデオタイトル-必須フィールド（最大256文字のシングルバイト文字）
poster 文字列 ヌル ポスター画像のURL
poster_sources 配列 [] ビデオのポスターに対応するオブジェクトの配列。現在、ポスターは1つしかないため、オブジェクトは1つです。オブジェクトには単一のプロパティがありますsrcその値はポスター画像のURLです。
published_at ISO 8601 日付時刻文字列 なし ビデオが公開された日時。例：「2014-12-09T 06:07:11 .877Z」
reference_id 文字列 ヌル アカウント内で一意の値（最大150文字のシングルバイト文字）
sources 配列 空の配列 の配列ビデオソースオブジェクト
tags 配列 空の配列 ビデオに追加されたタグ（文字列）の配列（タグあたり最大128個のシングルバイト文字）
text_tracks の配列text_trackオブジェクト 空の配列 ビデオに関連付けられたWebVTTキャプションのデータ
thumbnail 文字列 ヌル サムネイル画像のURL
thumbnail_sources 配列 [] ビデオのサムネイルに対応するオブジェクトの配列。現在、サムネイルは1つしかないため、オブジェクトは1つです。オブジェクトには単一のプロパティがありますsrcその値はサムネイル画像のURLです。
updated_at ISO 8601 日付時刻文字列 なし ビデオが最後に変更された日時。[1-2] 例：「2015-01-13T 17:45:21 .977Z」
variants 配列 [] ビデオの多言語メタデータを含むオブジェクトの配列

注

  • [1-2] 変更は、ユーザーまたはVideoCloudシステムによって行われた可能性があります

text_trackオブジェクト

表5：テキストトラックフィールド
フィールド タイプ デフォルト 説明
src 文字列（URL） ヌル .vttファイルが存在するURL
srclang ISO-639-1言語コード ヌル キャプションの言語
label 文字列 ヌル プレーヤーメニューで使用するラベル-キャプションと同じ言語である必要があります

注

  • [9-1] mime_typeの設定はオプションですが、キャプションはUTF-8でエンコードされ、mime_typeが「text / vtt」である必要があります。

key_systems物体

key_systemsオブジェクトには、DRMキー名ごとに1つずつ、オブジェクトのリストが含まれています。このオブジェクトが定義されている場合、そのソースはコンテンツで保護されています。

表4：key_systemsオブジェクト
フィールド タイプ デフォルト 説明
キー名 オブジェクト ヌル DRMタイプを指定するオブジェクトの名前–有効なキーは次のとおりです。

* フェアプレー：com.apple.fps.1_0
* PlayReady：com.microsoft.playready
*ワイドバイン：com.widevine.alpha
FairPlayはkey_request_urlとcertificate_url
他の人が使用するlicense_url		 文字列 ヌル DRMライセンスまたは証明書が存在する場所のURL
 

例1: 

    "key_systems": {
      "com.apple.fps.1_0": {
        "key_request_url": "https://....",
        "certificate_url": "https://..."
      }
    }

例2: 

    "key_systems": {
      "com.microsoft.playready": {
        "license_url": "https://..."
      },
      "com.widevine.alpha": {
        "license_url": "https://..."
      }
    },

variant object

各variant特定の言語のビデオメタデータのバージョンが含まれています。

表5：バリアントオブジェクト
フィールド タイプ デフォルト 説明
language 文字列 なし 言語-COUNTRYという形式のバリアントの言語。 ja-JA
name 文字列 なし 翻訳されたタイトル
description 文字列 なし 翻訳された説明
long_description 文字列 なし 翻訳されたlong_description
custom_fields オブジェクト {} カスタムフィールドのキーと値のペアのマップ

例

{
  "language": "ja-JA",
  "name": "バーゼル：ライン川",
  "description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "long_description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "custom_fields": {
    "language": "Japanese"
  }
}
