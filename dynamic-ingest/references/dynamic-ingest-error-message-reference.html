エラーメッセージ

次のエラーメッセージは、Dynamic Ingest API によって返すことができます

動的取り込みエラーメッセージ
コード メッセージ タイプ 説明
400 PROFILE json

名前でプロファイルが見つかりません
400 BAD_REQUEST json

リクエストボディを解析できません。
400 NOT_SUBMITTED json

ジョブを送信できません。後でもう一度試してください
400 NO_SUCH_VIDEO json

参照されているビデオが見つかりません
400 NO_SOURCE json

使用するソースが見つかりません
400 CDN_CREDENTIALS json

CDN 資格情報をフェッチできません
400 BAD_CALLBACKS json

コールバックが予想される形式ではありませんでした
401 UNAUTHORIZED json

認証に失敗しました。アクセストークンのクライアント資格情報が正しいことを確認してください
403 DYNAMIC_DELIVERY_NOT_ALLOWED json

このアカウントは動的配信では有効になっていませんが、動的配信プロファイルが指定されました
403 CONTEXT_AWARE_ENCODING_NOT_ALLOWED json

このアカウントはコンテキスト認識エンコーディングでは有効になっていませんが、コンテキスト認識エンコーディングプロファイルが指定されました
404 RESOURCE_NOT_FOUND json

APIはリクエストしたリソースを見つけられませんでした
415 UNSUPPORTED_MEDIA_TYPE json

取り込もうとしているファイルがサポートされているタイプではありません（を参照）サポートされているビデオおよびオーディオ形式、コーデック、およびコンテナ）またはリクエストに含まれていなかったContent-Type: application/jsonヘッダ
422 MALFORMED_SOURCE_URL json

ソース URL の形式が正しくありません。
422 BAD_PROTOCOL_SOURCE_URL json

ソース URL はサポートされていないプロトコルを使用しています。
422 EXCEED_MAXIMUM_VTT_SOURCES json

VTT ソースが最大サイズを超えています。
422 INVALID_VTT_KIND json

VTT の種類が無効です。
422 CONSTRAINT_VIOLATION json

イメージソースが提供されている場合、capture-image は使用できません。
422 UNPROCESSABLE_ENTITY json

リクエストデータには処理不能なエンティティが含まれています。
429 RATE_LIMIT_EXCEEDED json

動的取り込みジョブが作成されませんでした。再試行する前に、このアカウントの同時ジョブの数を減らしてください。このエラーはログに記録されました
400 CDN_CONFIGS json

CDN 資格情報をフェッチできません
400 AMBIGUOUS_REQUEST json

リクエストには、マスター URL と use_archived_master の両方が設定されています。
500 INGEST_ERROR json

このエラーは、通常、取り込みサービスが一時的に中断されたことを示します。
