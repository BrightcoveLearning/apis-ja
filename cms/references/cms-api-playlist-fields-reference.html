主フィールド

表1:[主フィールド]
フィールド タイプ 読み取り専用 デフォルト 説明
id 文字列 はい なし プレイリスト ID
account_id 文字列 はい なし アカウントの ID
created_at ISO 8601 日付時刻文字列 はい なし 日付時刻プレイリストがアカウントに追加されました。例:「2014-12-09T 06:07:11 .877Z」
description 文字列 いいえ ヌル プレイリストの簡単な説明-シングルバイト文字最大248文字
favorite ブール値 いいえ false プレイリストをお気に入りに含めるかどうか
limit 番号 いいえ 100年 再生リストが保持できる動画の数（整数、1〜100） スマートプレイリストの場合のみ返されます
name 文字列 いいえ なし プレイリストのタイトル-必須フィールド (最大長:248 文字のシングルバイト)
reference_id 文字列 いいえ ヌル アカウント内で一意な任意の値
search 文字列 いいえ ヌル スマートプレイリストの動画を定義するために使用される検索文字列 (例:「+タグ:「鳥」、「鳥」)
type 文字列 はい プレイリストの種類 ノートを参照してください[1] 未満
updated_at ISO 8601 日付時刻文字列 はい なし 日時プレイリストが最後に変更されました。[1-2] 例：「2015-01-13T 17:45:21 .977Z」
video_ids 配列 いいえ プレイリストの動画の ID の配列 手動プレイリストでのみ返される

注

  • [1]

    type値は次のいずれかになります。明示的ACTIVATED_OLDEST_TO_NEWEST、ACTIVATED_NEWEST_TO_OLDEST、ALPHABETICAL、PLAYS_TOTAL、PLAYS_TRAILING_WEEK、START_DATE_OLDEST_TO_NEWEST , START_DATE_NEWEST_TO_OLDEST

    EXPLICITは、手動プレイリストであることを意味します。他のすべてのタイプはスマートプレイリストで、その順序付けスキームに対応する名前の付いたリストです。
