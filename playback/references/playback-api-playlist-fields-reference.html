主フィールド

表1:[主フィールド]
フィールド タイプ デフォルト 説明
id 文字列 なし プレイリスト ID
account_id 文字列 なし アカウントの ID
created_at ISO 8601 日付時刻文字列 なし 日付時刻の動画がアカウントに追加されました。例:「2014-12-09T 06:07:11 .877Z」
description 文字列 null ビデオの簡単な説明-最大128のシングルバイト文字
favorite ブール値 false プレイリストをお気に入りに含めるかどうか
limit 番号 100 再生リストが保持できる動画の数（整数、1〜100） スマートプレイリストの場合のみ返されます
name 文字列 なし プレイリストのタイトル-必須フィールド
reference_id 文字列 null アカウント内で一意な任意の値
search 文字列 null スマートプレイリストの動画を定義するために使用される検索文字列 (例: +tags:"bird","birds" )
type 文字列 プレイリストの種類 ノートを参照してください[1] 未満
updated_at ISO 8601 日付時刻文字列 なし 日時プレイリストが最後に変更されました。例:「2015-01-13T 17:45:21 .977Z」
videos 配列 プレイリスト内のビデオのビデオオブジェクトの配列 再生リストには、最初の 20 本の動画のみが返されます。

注

  • [1]

    type値は次のいずれかになります。明示的ACTIVATED_OLDEST_TO_NEWEST、ACTIVATED_NEWEST_TO_OLDEST、ALPHABETICAL、PLAYS_TOTAL、PLAYS_TRAILING_WEEK、START_DATE_OLDEST_TO_NEWEST , START_DATE_NEWEST_TO_OLDEST

    EXPLICITは、手動プレイリストであることを意味します。他のすべてのタイプはスマートプレイリストで、その順序付けスキームに対応する名前の付いたリストです。
