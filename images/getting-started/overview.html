入門

Brightcove Image APIは、単純なGETリクエストを使用して、画像のサイズ変更、トリミング、回転をすばやく簡単に行う方法を提供します。

URL をリクエスト

ImageAPIのリクエストURLは次のとおりです。

https://images.brightcovecdn.com/image/v1/account_id/default/url/{image_url}

必要なパスパラメータは次のとおりです。

  • account_id：ビデオクラウドアカウントID
  • image_url：画像を見つけることができる公開URL
ノート：
  • 次の画像形式がサポートされています。PNG、JPG、GIF
  • NS image_url適切にエンコードする必要があります。JavaScriptを使用する場合は、encodeURIComponent()方法。また、使用することができます文字列エンコーダ Brightcove Learning Servicesが提供：

URL パラメータ

画像の変換バージョンを取得するには、リクエストで以下に説明する1つ以上のURLパラメータを使用します。3つのパラメータを任意に組み合わせて使用できることに注意してください。

ImageAPIのURLパラメータ
パラメーター 説明
resize これを「widthxheight」に設定して画像のサイズを変更します-例：サイズ変更= 160x90）
crop これを「widthxheight」に設定して、幅と高さを切り取ります-例：作物= 160x90 -原点は画像の中心です。
rotate 画像を指定された度数だけ時計回りに回転させます-例：回転= 90

ベストプラクティス

変換された画像がWebページまたはオンラインアプリに表示される場合は、APIを使用して画像を取得し、サーバーに保存します。行ういいえ APIのレート制限により、トラフィックの多いページやアプリで一部のリクエストが失敗するため、実行時にそれらを取得します。また、APIリクエストが読み込まれるたびに強制的に行われるため、ページ/アプリのパフォーマンスが低下します。ユーザー。
