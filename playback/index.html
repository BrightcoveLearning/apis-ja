--- title: Playback API parent: Home description: '

Playback API は、Video Cloud からビデオとプレイリストのデータを取得するために使用されます。通常、ウェブページまたはモバイルアプリでプレーヤーにビデオデータを取得するために使用されます。  Playback API は、Web ページやモバイルアプリから動画や再生リストのデータを取得するときのクライアント側での使用を目的とした、低レイテンシー API です。ですない Video CloudをCMSまたは他のシステムと統合する際に使用する一般的なメディア管理API-そのためには、 CMS API。

Playback API の認証にはポリシーキーが必要です。ポリシーキーは、Policy API を使用して、またはアカウントのプレーヤーから取得できます。
